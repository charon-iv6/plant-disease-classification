Welcome to Plant Disease Classification documentation!

This is a deep learning solution for classifying plant diseases into 16 classes, developed for a time-constrained classification challenge.

Quick Start

# Clone the repository
git clone https://github.com/charon-iv6/plant-disease-classification.git
cd plant-disease-classification

# Install dependencies
pip install -r requirements.txt

# Run the model
python src/final_model.py

Key Features

  • ResNet9 architecture optimized for plant disease classification

  • Mixed precision training for better performance

  • Test Time Augmentation (TTA)

  • Cross-platform support (Windows/macOS)

  • Bilingual support (English/Persian)
